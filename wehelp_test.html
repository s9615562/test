兩百字內簡介
我的名字是李佩穎，今年33歲,畢業於中山醫學大學語言治療與聽力學系，畢業後成為一名語言治療師，職業了4年多，目前待業中。 在職期間，發現治療師職業與自己的個性不符，待了幾年後也確定這份工作不適合自己，工作時會消耗大量能量，所以近一兩年一直有轉職的念頭。 在偶然間觀看到一位數據分析師的youtube工作紀錄影片，覺得這份工作內容很吸引我， 於是嘗試網路的免費教學影片，結果發現自己很享受寫程式思考的過程，決定繼續往軟體工程領域研究邁進。
為了成為軟體工程師，曾做過什麼努力？
幾乎看完彭彭老師youtube頻道的python課程，還成為頻道會員，學習mysql技能。後來因為補習班有企業媒合保證，對找工作有焦慮的我就報名了數據分析課程， 其中學了python基礎知識，也會爬取網路上的圖表和圖片，再用開檔寫檔的方式存入資料夾中。 但隨著自己接觸前端的內容越多，越發現前端比較偏向自己的興趣，可以看到畫面產出真的很有趣。但自己所學多為數據或後端相關，跟前端使用的程式語言較無關聯。 於是我開始自學前端語言，學習頻道包含彭彭老師的youtube頻台以及w3 school的學習網站，目前還在學習基本的html、css、javascript等。也在努力寫出小作品。 這時剛好看到彭彭老師的營隊計畫-計畫包含培養獨立學習能力與開發專案的方向引導，覺得這些宗旨跟我的觀念很契合! 之後如果有幸可以成為計畫學員，我也會繼續在開課前加強自己欠缺的技能，例如:繼續熟練html開發小作品或學習架設flask等等，讓自己可以順利追上每週學習進度。
如果參與這個計畫，會怎麼安排學習時間？
因為知道這個計畫的挑戰性，需要完全的投入與健康身體，所以之後會全職參與計畫，來完成每週任務。 預計周一至周五每天大約八小時學習，若進度沒追上，則增加週六繼續完成。週日為彈性休息日，如果時間允許，會安排爬山或健身等增加體能的活動。 在找到軟體工作前因為沒有經濟壓力，所以也不打算兼職上班，我認為這樣較可以讓我全心投入於寫程式中。
是否有想要加入的軟體公司？為什麼想加入該公司？
目前還沒有特別想要加入的公司，不過因為之後會定居台南，所以會希望加入台南地區的軟體公司或是可以遠端上班的公司。
請描述一件你曾經碰到最困難的事情，如何克服？
之前的職業是語言治療師，所以很常碰到需要哄小孩的時候，但偏偏我比較沒有小孩緣，尤其聰明黏媽媽的小孩，只要單獨跟我上課常常會哭鬧整節課，某位給我治療的一歲多小男孩就這樣哭哭停停持續了幾個月。 為了順利上課，我看了如何和小孩互動的書籍，甚至全英文國外語言治療師的治療影片，雖然我覺得自己有進步，但對於那位一歲多小男孩還是效果有限。 某次在治療小男孩時，前輩學姊剛好進教室拿東西，學姊知道了我的狀況後，馬上示範了他如何處理，神奇的是弟弟居然不哭了。學姊表示這不是什麼特殊手法，只是經驗，要我不要太在意。我之後持續使用該方法，發現對大部分小孩都適用。 所以我覺得遇到困難時，一開始可以先查詢網路資源，在一般情況下這可以解決大部分難題，但如果還是無法解決，則可以詢問同儕或資深前輩，或許只是一個方向的引導，但確也是最有效率的!
關於這份申請網頁，請分享一個你開發時的技術心得。
一開始一直卡在 GitHub Pages的使用，雖然彭彭老師有給予教學影片提示，但因為 GitHub Pages後來似乎有改版，所以我一直不知道如何找到屬於我自己的網址。後來我陸陸續續觀看了其他較近期的 GitHub Pages的教學影片， ，並拼湊了每個人的方法，最後總算成功的產出了我的第一個網頁，看到現在的內容成功在網頁上穩當的呈現，真的非常感動。
其他想要對我們說的事情？
我知道轉職是條辛苦的路，就算轉職成功也不是輕鬆的開始，而是更艱鉅的挑戰。我覺得習慣於解決問題才有辦法可以持續學習寫程式。 也因為我嘗試過其他行業，所以知道面對挑戰比待在不適合的原地更幸福。我很幸運有這樣的時間金錢還有支持我的家人，讓我可以面對我想要的挑戰。 希望之後有機會成為WeHelp的一份子，有很多夥伴和助教陪伴，一定比較不孤單。感謝閱讀完我的申請表!
